# Gmail-Growl Chocolatey Package

**Status:** Deprecated
**Version:** 2.4.1

## Description

Gmail Growl monitors a Gmail account for new mails and displays fancy desktop alerts.

**Note:** This package is **deprecated** as the upstream project is retired. No software will be installed when running this package.

Since this package is deprecated, installing it will only display a warning message:

WARNING: This package is deprecated as the upstream project is retired. No software will be installed.
